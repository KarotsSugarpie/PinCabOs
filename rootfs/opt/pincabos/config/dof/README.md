# PinCabOS DOF config

Ce dossier contient la configuration DOF utilisée par PinCabOS.

Fichiers possibles :
- `directoutputconfig.ini`
- `directoutputconfig30.ini`
- `directoutputconfig90.ini`
- `tablemappings.xml`
- `GlobalConfig_B2SServer.xml`
- `DirectOutputShapes.xml`
- `cabinet.xml` ou une config cabinet générée localement
- `configtool-api-key.txt` : clé API locale, NE PAS versionner avec une vraie clé

Notes :
- Les fichiers DOF téléchargés depuis DOF Config Tool peuvent être placés ici.
- Les configs spécifiques au hardware du cab doivent être régénérées/importées localement.
